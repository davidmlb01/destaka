## Regras de Copy (inegociáveis)

- NUNCA usar "otimizar" ou "otimização" — trocar por: cuidar, arrumar, colocar em ordem, resolver
- NUNCA usar prova social até ter depoimentos reais de clientes
- NUNCA usar jargão técnico (algoritmo, SEO, SEO local) — linguagem de dentista, não de dev
- NUNCA overpromise: não prometer número de pacientes, prometer visibilidade e métricas concretas
