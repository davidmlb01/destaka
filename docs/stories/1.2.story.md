# Story 1.2 — Diagnóstico & Score Engine
**Epic:** 1 — Core Platform
**Status:** Planejado
**Prioridade:** P0
**Estimativa:** 4 dias
**Depende de:** Story 1.1

## Contexto
O coração da plataforma. Lê o perfil completo via API, calcula o score 0-100 por categoria e usa Claude para gerar um diagnóstico em português com linguagem acessível para o profissional.

## Acceptance Criteria

- [ ] Job assíncrono `full_diagnostic_job` executa após conexão
- [ ] Busca todos os dados do perfil via Google Business Profile API
- [ ] Score calculado por 6 categorias (algoritmo da arquitetura)
- [ ] Score total 0-100 calculado corretamente
- [ ] Claude gera diagnóstico em PT-BR específico por segmento
- [ ] Diagnóstico salvo em `diagnostics` no banco
- [ ] E-mail enviado ao usuário quando diagnóstico fica pronto
- [ ] Diagnóstico exibe no dashboard em cards por categoria
- [ ] Cada card mostra: score, o que está faltando, impacto estimado
- [ ] Tempo total do job: < 30 segundos

## Tasks

- [ ] Implementar `GoogleBMPClient` (wrapper da API com retry + rate limit)
- [ ] Implementar `ScoreCalculator` (algoritmo de pontuação por categoria)
- [ ] Implementar `DiagnosisService` (orquestra leitura + score + Claude)
- [ ] Implementar prompt Claude para diagnóstico por segmento
- [ ] Implementar job BullMQ: `full_diagnostic_job`
- [ ] Criar worker para processar a fila
- [ ] Implementar e-mail "Diagnóstico Pronto" (Resend)
- [ ] UI: cards de score por categoria no dashboard
- [ ] UI: diagnóstico em texto (collapsible)
- [ ] Testes unitários: ScoreCalculator com fixtures de perfis reais
