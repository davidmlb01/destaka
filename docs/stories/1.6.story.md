# Story 1.6 — Gestão de Avaliações
**Epic:** 1 — Core Platform
**Status:** Planejado
**Prioridade:** P1
**Estimativa:** 3 dias
**Depende de:** Story 1.1

## Contexto
Avaliações são o fator mais poderoso no rankeamento local. A plataforma lista todas as avaliações, gera respostas inteligentes via Claude e permite publicar com 1 clique.

## Acceptance Criteria

- [ ] Listagem de todas as avaliações (com paginação)
- [ ] Filtros: todas / sem resposta / negativas
- [ ] Para cada avaliação sem resposta: botão "Gerar Resposta"
- [ ] Claude gera resposta contextualizada (segmento + tom profissional)
- [ ] Usuário pode editar a resposta antes de publicar
- [ ] Publicação via Google Business Profile API com 1 clique
- [ ] Indicador visual: respondida / não respondida / negativa
- [ ] Alerta no dashboard: "X avaliações aguardando resposta"
- [ ] Histórico de respostas publicadas

## Tasks

- [ ] Implementar `ReviewService` (sync de avaliações via API)
- [ ] Job de sync de avaliações (a cada 6h)
- [ ] Prompt Claude para resposta por segmento e rating
- [ ] Endpoint POST /api/reviews/{id}/reply
- [ ] UI: lista de avaliações com stars e estado
- [ ] UI: modal de edição/publicação de resposta
- [ ] UI: badge de alerta no dashboard
- [ ] Testes: geração de respostas para diferentes ratings e segmentos

## Notas Técnicas
- Resposta para 1 estrela: tom empático, não defensivo, convite para contato direto
- Resposta para 5 estrelas: agradecimento personalizado, menção ao serviço se disponível
- Máximo 4.096 caracteres por resposta (limite da API)
