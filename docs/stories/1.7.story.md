# Story 1.7 — Posts Automáticos Semanais
**Epic:** 1 — Core Platform
**Status:** Planejado
**Prioridade:** P1
**Estimativa:** 3 dias
**Depende de:** Story 1.1

## Contexto
Postar semanalmente aumenta impressões em 26%. Claude gera posts relevantes e localizados para cada segmento. É o principal argumento para a assinatura recorrente.

## Acceptance Criteria

- [ ] Cron job toda segunda-feira às 9h gera e publica post
- [ ] Post gerado por Claude: específico para segmento + sazonalidade brasileira
- [ ] Modos: automático (publica direto) ou aprovação (envia para revisar)
- [ ] Usuário escolhe modo nas configurações
- [ ] Post tem: texto (300 chars max) + CTA + tipo (UPDATE/OFFER/EVENT)
- [ ] Histórico de posts publicados visível no dashboard
- [ ] Preview do próximo post agendado
- [ ] E-mail de aprovação (modo aprovação) com link para aprovar/editar
- [ ] Usuário pode criar post manual a qualquer momento

## Tasks

- [ ] Prompt Claude para posts por segmento + contexto sazonal
- [ ] Cron job: `weekly_post_job` (todo domingo às 22h para publicar segunda)
- [ ] Implementar modo automático e modo aprovação
- [ ] Endpoint POST /api/posts/generate (post manual)
- [ ] Endpoint POST /api/posts/{id}/publish
- [ ] E-mail de aprovação com botões de ação
- [ ] UI: seção de Posts no dashboard
- [ ] UI: histórico + próximo post agendado
- [ ] UI: criador de post manual
