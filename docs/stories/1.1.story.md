# Story 1.1 — Google OAuth + Conexão GMB
**Epic:** 1 — Core Platform
**Status:** Planejado
**Prioridade:** P0 (bloqueante para tudo)
**Estimativa:** 3 dias

## Contexto
Ponto de entrada da plataforma. Sem OAuth funcionando, nada mais funciona. O usuário precisa autorizar o GMM a acessar seu Google Business Profile.

## Acceptance Criteria

- [ ] Botão "Conectar com Google" na landing/onboarding
- [ ] Fluxo OAuth 2.0 completo com Google
- [ ] Scopes solicitados: `business.manage`, `email`, `profile`
- [ ] Access token e refresh token salvos criptografados no banco
- [ ] Após auth: listar todos os perfis GMB do usuário
- [ ] Usuário seleciona qual perfil gerenciar
- [ ] Perfil salvo em `gmb_profiles` com dados básicos
- [ ] Redirecionamento para dashboard após seleção
- [ ] Tratamento de erro: OAuth negado pelo usuário
- [ ] Tratamento de erro: usuário sem perfil GMB

## Tasks

- [ ] Configurar next-auth v5 com provider Google
- [ ] Configurar escopos OAuth no Google Cloud Console (documentar)
- [ ] Implementar endpoint GET /api/gmb/locations (lista perfis)
- [ ] Implementar UI de seleção de perfil (modal pós-auth)
- [ ] Implementar criptografia AES-256-GCM para tokens
- [ ] Salvar perfil selecionado em gmb_profiles
- [ ] Disparar job de diagnóstico após seleção
- [ ] Testes: fluxo happy path + error states
