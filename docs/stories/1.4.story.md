# Story 1.4 — Checklist Guiado (Ações Manuais)
**Epic:** 1 — Core Platform
**Status:** Planejado
**Prioridade:** P1
**Estimativa:** 3 dias
**Depende de:** Story 1.2

## Contexto
Tudo o que a API não consegue fazer automaticamente precisa de ação do usuário. A plataforma guia o usuário passo a passo, com instruções claras e exemplos visuais.

## Acceptance Criteria

- [ ] Checklist priorizado (P0 → P1 → P2) exibido no dashboard
- [ ] Cada item tem: título, descrição clara, impacto no score, instrução passo-a-passo
- [ ] Fotos: guia visual do que fotografar (ex: recepção, equipe, procedimentos)
- [ ] Verificação do perfil: instruções se ainda não verificado
- [ ] Usuário pode marcar item como "Feito"
- [ ] Score projeta melhoria após cada item concluído
- [ ] E-mail de lembrete semanal para itens pendentes
- [ ] Progresso salvo no banco

### Itens do checklist V1:
- [ ] Adicionar 5+ fotos do espaço físico
- [ ] Adicionar foto de perfil da clínica/consultório
- [ ] Adicionar foto da equipe
- [ ] Verificar o perfil (se pendente)
- [ ] Solicitar avaliações a clientes recentes
- [ ] Responder avaliações em aberto

## Tasks

- [ ] Implementar `ChecklistGenerator` (gera itens a partir do diagnostic)
- [ ] UI: lista de tarefas com accordion para instruções detalhadas
- [ ] UI: upload de fotos com guia visual (orientações por segmento)
- [ ] UI: projeção de score ao completar tarefas
- [ ] Implementar marcação de tarefa como concluída
- [ ] Implementar cron job de e-mail de lembrete (toda segunda-feira)
